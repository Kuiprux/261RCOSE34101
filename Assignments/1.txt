운영체제 실습 과제 1

윈도우 환경에서 VirtualBox를 설치하였으며, Ubuntu LTC 24.04 가상 머신을 생성하였습니다.

1. 실습 내용 조사
apt는 advanced packaging tool을 의미하며, 패키지의 설치나 삭제, 업그레이드 등을 관리해 주는 도구이다.
1. sudo apt update: 패키지의 정보가 담긴 index file을 동기화한다. 새로운 버전 정보 등을 받아올 수 있다.
2. sudo apt upgrade: 업데이트된 새로운 버전 정보를 기반으로 패키지의 버전을 업그레이드한다.
3. mkdir /home/<본인이니셜>/<guestOS공유폴더이름>: 지정된 경로에 폴더를 생성한다. 여기서는 <이니셜> 사용자의 폴더 안에 공유 폴더로 사용할 폴더를 생성하였다.
4. echo "sudo mount -t 9p -o trans=virtio share /home/<본인이니셜>/<guestOS공유폴더이름> -oversion=9p2000.L" >> /home/<본인이니셜>/.profile: 큰따옴표 안에는 VirtualBox 설정에서 지정한 공유 폴더를 앞서 생성한 폴더에 마운트하는 명령어가 있으며, <이니셜> 사용자의 .profile 파일에 덧붙임으로서 해당 사용자가 로그인 시마다 공유 폴더를 마운트할 수 있게끔 한다.

2. 공유 폴더 생성과 파일 생성
공유 폴더를 설정하여 가상 머신 내의 폴더에 마운트하였습니다. 가상 머신에서 cd로 폴더 이동 후 cat > shared_file.txt로 파일 내용을 작성하였습니다.
